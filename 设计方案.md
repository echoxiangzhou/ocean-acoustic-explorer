## 系统定位

**OceanAcoustic Explorer** — 面向海洋声学研究和军事应用的交互式声场特征可视化平台，作为 OceanAgent 平台的声学分析子模块，支持全球尺度浏览和区域精细分析两种模式。

------

## 一、总体架构

**前后端分离 + 微服务计算引擎**

| 层级         | 技术选型                                                     | 职责                                      |
| ------------ | ------------------------------------------------------------ | ----------------------------------------- |
| 前端展示层   | React + CesiumJS（三维地球）+ Deck.gl（高性能图层）+ ECharts（图表） | 地图交互、剖面图、声线图、TL 伪彩色图渲染 |
| API 网关     | FastAPI                                                      | 路由分发、鉴权、请求缓存                  |
| 声学计算服务 | Python 微服务（arlpy/Bellhop + pyram + gsw）                 | 声速计算、声线追踪、传播损失建模          |
| 数据服务     | xarray + Zarr + TileDB                                       | WOA23/HYCOM/SODA/GEBCO 数据切片与按需供给 |
| 缓存层       | Redis                                                        | 高频查询结果缓存（如 WOA 气候态声速场）   |
| 存储层       | MinIO / 本地 Zarr Store                                      | 预计算产品和原始数据统一管理              |

------

## 二、功能模块设计

### 模块 A：全球声场特征总览

**视图形式**：二维/三维地球底图 + 热力图叠加

**可切换图层（6 层）**：

1. 声道轴深度分布（色阶 0–1500m）
2. 表面声道厚度分布（色阶 0–300m，无表面声道区域灰显）
3. 主跃层声速梯度强度（色阶 dB/m）
4. 第一会聚区距离（色阶 30–70km）
5. 声影区起始距离（给定声源深度）
6. 声场类型分类（深海CZ型 / 浅海波导型 / 极地声道型 / 混合型，离散色块）

**交互能力**：

- 时间滑块控制月份（1–12 月），图层随季节动态切换，支持自动播放动画
- 声源深度滑块（0–500m），实时更新会聚区距离和声影区图层
- 频率选择器（50 / 100 / 500 / 1000 / 5000 Hz），影响声场类型判定
- 鼠标悬停任意网格点，弹出 tooltip 显示该点全部 6 项特征值
- 点击任意位置进入"区域精细分析"（模块 C）

**数据源**：WOA23 预计算产品，前端加载 Zarr 瓦片

------

### 模块 B：声速剖面浏览器

**视图形式**：左侧地球/地图选点，右侧声速剖面曲线图

**核心功能**：

- 点击地图任意位置，右侧即时绘制该点声速剖面 c(z) 曲线
- 支持多点对比：最多同时叠加 5 条剖面，用不同颜色区分
- 剖面图上自动标注：声道轴位置（水平虚线）、表面声道层（浅蓝色底）、主跃层区间（渐变色底）、海底深度（灰色填充）
- 月份切换：观察同一位置声速剖面的季节变化，支持 12 个月动画循环
- 数据源切换：WOA23 气候态 / HYCOM 实时场 / SODA 历史场，对比不同数据源的差异

**附加面板**：

- 特征值卡片：声道轴深度、声道轴声速、表层声速、Δc、表面声道厚度
- 声速公式选择：Mackenzie / Chen-Millero / TEOS-10，可切换对比

**数据源**：WOA23（默认）/ HYCOM（按需请求后端计算）

------

### 模块 C：区域精细声场分析

**视图形式**：用户在地图上画线定义断面 → 后端计算 → 返回声场截面图

**交互流程**：

1. **定义断面**：在地图上点击两个端点画一条线，或从预置断面列表选择（南海 18°N、吕宋海峡、西太 130°E 等）

2. 参数设置面板

   ：

   - 声源深度（拖拽滑块，0–1000m）
   - 声源频率（下拉选择 + 自定义输入）
   - 计算模型（Bellhop 射线 / RAM 抛物方程 / Kraken 简正波）
   - 计算距离范围（自动根据断面长度设定，可手动调整）
   - 海底参数（自动从 deck41 查询，允许手动覆盖）

3. **提交计算**：点击"Run"，后端异步执行，前端显示进度条

4. 结果展示

   （多标签页）：

   - **传播损失 TL(r,z) 伪彩色图**：横轴距离、纵轴深度，Jet/Viridis 色阶，叠加海底地形轮廓线。用半透明标注框标识会聚区（CZ）、声影区（SZ）、底反射区（BB）
   - **声线图**：Bellhop 射线轨迹，可选显示/隐藏不同出射角范围的声线，用颜色编码声线强度
   - **断面声速场**：该断面的声速 c(r,z) 分布，作为声场解释的辅助图
   - **定深 TL 曲线**：用户拖拽一条水平线选择接收深度，实时绘制 TL vs Range 曲线，会聚区表现为周期性凹陷
   - **定距 TL 曲线**：用户拖拽一条垂直线选择距离，实时绘制 TL vs Depth 曲线，声影区表现为能量骤降

**数据源**：HYCOM（T/S）+ GEBCO（地形）+ deck41（底质），后端实时计算

------

### 模块 D：时变趋势分析

**视图形式**：时间序列仪表盘

**功能**：

- 选择全球任意网格点或预定义区域（南海、北大西洋、西太等）
- 展示 1980–2022 年声场特征时间序列：
  - 声道轴深度年际变化
  - 表面声道季节窗口长度（每年有几个月存在表面声道）
  - 会聚区距离年际变化
  - Δc（表层-声道轴声速差）趋势
- 叠加线性趋势拟合线和置信区间
- 支持与 SST、ENSO 指数等气候指数的联动对比

**数据源**：SODA 3.15.2 预计算产品

------

### 模块 E：场景模拟器（高级功能）

**用途**：面向军事/工程应用的"假设分析"工具

**功能**：

- 用户设定场景：声源类型（点源/线源）、声源深度、接收器深度、频率、海域
- 自动加载该海域当月声速剖面和地形
- 实时计算并展示：探测距离估算、最优接收深度建议、会聚区利用方案
- 支持"对比模式"：夏季 vs 冬季、浅声源 vs 深声源、不同频率

## 三、UI/UX 设计方向

**风格定位**：深色科技风（Deep Ocean Theme）— 以深海蓝黑为主色调，数据可视化用高对比色阶，营造专业海洋科研仪器的沉浸感。

**配色体系**：

- 背景：`#0a1628`（深海蓝黑）
- 面板：`#111d33`（深蓝灰），`rgba(255,255,255,0.06)` 边框
- 主强调色：`#00d4ff`（海洋青）用于交互元素和高亮
- 辅助强调色：`#ff6b35`（暖橙）用于警告和关键标注
- 数据色阶：Viridis / Turbo（传播损失）、RdYlBu_r（声速场）

**布局**：

- 顶部：极简导航栏，模块切换标签（A/B/C/D/E）
- 左侧：参数控制面板（可折叠），宽度约 320px
- 中央：主可视化区域（地图 / 剖面图 / TL 图），占满剩余空间
- 右侧：信息面板 / tooltip / 特征值卡片（按需弹出）
- 底部：时间轴滑块（全局月份/年份控制）

**交互原则**：

- 所有参数变更即时响应（WOA 预计算数据），重计算任务显示进度
- 地图上的 hover → tooltip，click → 深入分析，双击 → 全屏该区域
- 图表支持导出 PNG/SVG，数据支持导出 CSV/NetCDF
- 响应式设计兼顾 1920×1080 桌面和平板

## 四、数据流与性能策略

**预计算 vs 实时计算的分界**：

| 内容             | 策略                             | 响应时间目标 |
| ---------------- | -------------------------------- | ------------ |
| 全球声场特征图层 | 预计算为 Zarr 瓦片，前端直接加载 | < 1 秒       |
| 声速剖面查询     | 预计算 WOA 声速场，按坐标索引    | < 0.5 秒     |
| HYCOM 剖面查询   | 后端按需从 OPeNDAP/本地缓存提取  | 2–5 秒       |
| 断面传播损失计算 | 后端实时调用 Bellhop/RAM         | 30 秒–3 分钟 |
| SODA 趋势查询    | 预计算时间序列，按坐标索引       | < 1 秒       |

**性能优化手段**：

- 全球图层使用多级金字塔瓦片（类似地图瓦片），缩放时按需加载分辨率
- 高频查询结果（热门海域声速剖面）缓存至 Redis，TTL 24 小时
- 断面计算采用 WebSocket 推送进度，避免长轮询
- 前端 TL 伪彩色图用 WebGL 渲染（Deck.gl BitmapLayer），避免 Canvas 2D 性能瓶颈
